## ðŸ›  Usage

Import the component and use it in your Svelte application:

```Svelte
<script>
  import Gauge from "svelte-gauge";
</script>

<Gauge
  value={value}
  start={0}
  stop={100}
  stroke={20}
  color={"red"}
  labels={["One", "Two", "Three"]}
  segments={[
    [0, 33],
    [34, 66],
    [67, 100],
  ]}
  width="300px"
  class="custom-gauge"
/>

```
